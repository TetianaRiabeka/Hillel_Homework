Картинки

Список команд

  1. git status - показує в яких файлах були внесені зміни
  2. git add . - підготувати ці файли до комміту
  3. git branch - перевірити які гілки є і яка є активна
  4. git branch lesson1 - створено нову гілку lesson1
  5. git branch -d lesson1 - видалення гілки lesson1
  6. git pull - отримати зміни з серверу
  7. git push - переказує зміни із серверу
  8. git log - показує кто і коли вносив зміни
  9. git commit -m text - створюється комміт, який показує які зміни там відбулися
  10. git remote -v - лінк на репозиторій
  11. git checkout -b lesson1 - створює нову гілку з назвою lesson1 і до неї переходить
  12. git checkout lesson1 - перехід на гілку lesson1
  13. git reset - відміняє кількість коммітів яка нам потрібна
  14. git push origin HEAD - переказ змін до репозиторію origin до поточної гілки в якій ми зараз знаходимося
  15. git pull origin HEAD - отримати зміни з серверу з репозиторію origin
  16. git rebase - позволяє повернутись від чогось назад
  17. git rebase HEAD-1 - позволяє повернутись на 1 минулий комміт
